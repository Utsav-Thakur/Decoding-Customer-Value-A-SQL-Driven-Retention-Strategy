## ✦ Core Features & Modules

### 1. Split-Screen Hero Executive Dashboard
- Immersive high-contrast luxury dashboard (burgundy, gold, and cream).
- Live KPI counter stats (Total customers, AOV, and Promo exposure) using custom `requestAnimationFrame` animations.
- Real-time animated **Value Pyramid Composition** donut chart with interactive hover tooltips.

### 2. Segment Value Pyramid
- Visualizes customers across 4 quartiles: **Platinum**, **Gold**, **Silver**, and **Bronze** cohorts.
- High-priority operational playbooks mapped to each customer segment.

### 3. Promo Dependency Diagnostics
- **AOV Analysis**: Compares basket value differences between promotion-exposed and full-price shoppers.
- **Promo Dependency vs Spend Scatter**: Identifies promo-trapped customers to mitigate margin leakage.

### 4. Regional Opportunity Map
- Maps customer concentration and opportunity density across US States to identify organic brand pull.

### 5. Category Funnel Journey
- Traces customer acquisition flow from initial entry categories (e.g. Clothing) to high-retention categories (e.g. Outerwear).

### 6. Ideal Customer Profile (ICP) Radar
- Dynamic comparison radar mapping properties of your ideal, full-price VIP customer profile against standard portfolio performance.

### 7. JARVIS — Built-in Offline Strategist
- An intelligent assistant that runs entirely locally with zero external API requirements.
- Equipped with **"What does this mean?"** triggers linked to every chart for instant, contextual business and data interpretations.

## 🛠 Tech Stack & Setup

- **Frontend:** React 19, Recharts (v3), Lucide React
- **Build Engine:** Vite, Tailwind CSS (v4)

### Local Development Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the local server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
